## Swapping in the real screenshots

The SVGs for the four `collab-*` files are stylized recreations of the product UI.
To use the actual screenshots:

1. Drop the PNG into this folder using the **same base name** (e.g. `collab-deck-full.png`).
2. In the root `README.md`, change that image's extension from `.svg` to `.png`.

Drag-and-drop into this folder on GitHub preserves filenames.
