# Defender KQL catalog

Two kinds of query: **discovery** (find *new* candidate IOCs — runs first, seeds
your `config.json`) and **targeted hunt** (search for *known* IOCs split by
machine type — generated by `gen_queries.py`).

---

## Phase 0 — Discovery (seed the IOC list)

These surface anomalies — things that appeared *yesterday* but never in the prior
window — so you can copy the interesting ones into `config.iocs`. Run them in
Defender Advanced Hunting, eyeball the results, and lift the suspicious
domains/hashes into your config.

### D1 — New base domains contacted yesterday (not seen in 90d)
Parametrize: `ago(90d)` look-back, `RequestsYesterday < N` noise cap.

```kusto
DeviceNetworkEvents
| where Timestamp >= ago(90d)                       // history window
| where isnotempty(RemoteUrl)
| extend Host = tolower(tostring(parse_url(RemoteUrl).Host))
| where isnotempty(Host)
| where Host matches regex @"[a-zA-Z]"
| extend Parts = split(Host, ".")
| extend PartsCount = array_length(Parts)
| extend BaseDomain = iff(
      PartsCount >= 2,
      strcat(tostring(Parts[PartsCount - 2]), ".", tostring(Parts[PartsCount - 1])),
      Host)
| extend Period = case(
      Timestamp >= startofday(ago(1d)) and Timestamp < startofday(now()), "yesterday",
      "history")
| summarize RequestsYesterday = countif(Period == "yesterday"),
            RequestsBefore    = countif(Period == "history")
        by BaseDomain
| where RequestsYesterday > 0 and RequestsBefore == 0    // brand-new domain
| where RequestsYesterday < 2                            // low-volume = stealthy; raise to widen
| order by RequestsYesterday asc
```

> The `BaseDomain` reduction (eTLD+1 by naive last-two-labels) is good enough for
> triage; it over-collapses multi-label TLDs like `co.uk` — accept that for speed.

### D2 — New executables created yesterday (not seen in 90d)
Parametrize: device prefix (`IL-`), folder exclusions, look-back, noise cap.

```kusto
DeviceFileEvents
| where tolower(FileName) matches regex @"\.(exe|msi|bat|cmd|ps1)$"
| where DeviceName startswith "IL-"                  // <- your device naming prefix
| where Timestamp >= ago(90d)
| where isnotempty(SHA256)
| where ActionType == "FileCreated"
// ----- folder allow-list exclusions (trusted build/install paths) -----
| where not(tolower(FolderPath) startswith @"c:\windows")
| where not(tolower(FolderPath) startswith @"c:\program files")
| where not(tolower(FolderPath) startswith @"c:\program files (x86)")
| where not(tolower(FolderPath) startswith @"c:\projects\scexpert")
| where not(tolower(FolderPath) startswith @"c:\repos\scexpert4")
// ----------------------------------------------------------------------
| extend Period = case(
      Timestamp >= startofday(ago(1d)) and Timestamp < startofday(now()), "yesterday",
      "history")
| summarize FilesYesterday = countif(Period == "yesterday"),
            FilesBefore    = countif(Period == "history"),
            FileNames = make_set(FileName, 5),
            Devices   = make_set(DeviceName, 5),
            Folders   = make_set(FolderPath, 5)
        by SHA256
| where FilesYesterday > 0 and FilesBefore == 0
| where FilesYesterday < 2
| order by FilesYesterday asc
```

> Lift the resulting `BaseDomain` values and `SHA256` values into
> `config.iocs.domains` / `config.iocs.hashes`, then move to the targeted hunt.

## Phase 1/2/3 — Targeted hunt (one query per machine group)

This is the template `scripts/gen_queries.py` fills from `config.json`. It splits
by `machine_groups[].filter` and injects the IOC lists. You normally don't write
this by hand — regenerate from config instead.

```kusto
let LookBack = {time_window_days}d;
let SuspectDomains = dynamic([ ...config.iocs.domains... ]);
let SuspectUrls    = dynamic([ ...config.iocs.urls... ]);
let SuspectHashes  = dynamic([ ...config.iocs.hashes... ]);
let NetworkHits =
    DeviceNetworkEvents
    | where Timestamp > ago(LookBack)
    | where RemoteUrl has_any (SuspectDomains)
         or RemoteUrl has_any (SuspectUrls)
         or RemoteIP in (SuspectDomains)
    | extend Ioc = coalesce(RemoteUrl, RemoteIP), IocType = "network"
    | project Timestamp, DeviceId, DeviceName, OSPlatform, DeviceType,
              AccountName = InitiatingProcessAccountName, Ioc, IocType,
              RemoteIP, SHA256 = InitiatingProcessSHA256,
              ProcessName = InitiatingProcessFileName;
let FileHits =
    DeviceFileEvents
    | where Timestamp > ago(LookBack)
    | where SHA256 in~ (SuspectHashes) or SHA1 in~ (SuspectHashes) or MD5 in~ (SuspectHashes)
    | extend Ioc = coalesce(SHA256, SHA1, MD5), IocType = "file"
    | project Timestamp, DeviceId, DeviceName, OSPlatform, DeviceType,
              AccountName = InitiatingProcessAccountName, Ioc, IocType,
              RemoteIP = "", SHA256, ProcessName = FileName;
union NetworkHits, FileHits
| where {machine_group.filter}        // e.g. DeviceType == "Server"
| extend MachineGroup = "{machine_group.name}"
| project Timestamp, DeviceName, DeviceId, MachineGroup, OSPlatform, DeviceType,
          AccountName, Ioc, IocType, RemoteIP, SHA256, ProcessName
| sort by Timestamp desc
```

Export each group's results to `./exports/defender/` as CSV, then run
`scripts/enrich.py`.
