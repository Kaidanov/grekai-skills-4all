# daily-cv Config

This directory contains user-editable setup files for the `daily-cv` skill.

- `SOURCES.md` defines private user-owned profile, tracker, and storage settings.
- `JOB_CRITERIA.md` defines target roles, geography, work-mode rules, compensation, and disqualification rules.
- `WATCHLIST.md` defines companies, ATS boards, community sources, and proactive outreach targets.

Keep public forks free of private links, contact details, personal trackers, and generated application files.
